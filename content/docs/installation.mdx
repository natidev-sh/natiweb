# Installation

Get started with Nati.dev in minutes.

## Desktop App

### macOS

1. Download the latest `.dmg` file from [nati.dev/download](https://nati.dev/download)
2. Open the `.dmg` file
3. Drag Nati to your Applications folder
4. Launch Nati from Applications

```bash
# Or install via Homebrew
brew install --cask nati
```

### Windows

1. Download the latest `.exe` installer from [nati.dev/download](https://nati.dev/download)
2. Run the installer
3. Follow the installation wizard
4. Launch Nati from Start Menu

```powershell
# Or install via winget
winget install Nati.Nati
```

## System Requirements

- **macOS**: 10.15 (Catalina) or later
- **Windows**: Windows 10 or later
- **RAM**: 4GB minimum, 8GB recommended
- **Disk Space**: 500MB for app, additional space for AI models

## First Launch

On first launch, you'll be guided through:

1. **API Key Setup** - Add your AI provider API keys
2. **Profile Creation** - Set up your account (optional)
3. **Theme Selection** - Choose light or dark mode

## Updating

Nati automatically checks for updates. When a new version is available:

1. You'll see a notification
2. Click "Download Update"
3. Restart the app to install

## Troubleshooting

### App won't open on macOS

```bash
xattr -cr /Applications/Nati.app
```

### Permission issues on Windows

Run as administrator:
1. Right-click Nati icon
2. Select "Run as administrator"
