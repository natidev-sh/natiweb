# Getting Started

Build your first AI-powered application in minutes.

## Step 1: Configure API Keys

1. Open Nati.dev
2. Navigate to **Settings â†’ API Keys**
3. Add your API keys:

```bash
OpenAI API Key: sk-...
Anthropic API Key: sk-ant-...

```

## Step 2: Create Your First App

1. Click **"New App"** from the home screen
2. Choose a template or start from scratch
3. Give your app a name and description
4. Click **"Create"**

### Available Templates

- **Chat Assistant** - Build a conversational AI
- **Code Generator** - Generate code from descriptions
- **Content Writer** - Create blog posts and articles
- **Data Analyzer** - Analyze and visualize data

## Step 3: Configure Your App

### Choose Your Model

Select from available AI models:

```typescript
// OpenAI Models
- GPT-4 Turbo
- GPT-4
- GPT-3.5 Turbo

// Anthropic Models
- Claude 3 Opus
- Claude 3 Sonnet
- Claude 3 Haiku

// Google Models
- Gemini Pro
- Gemini Ultra
```

### Set System Prompt

Define how your AI should behave:

```
You are a helpful assistant that provides clear, 
concise answers to user questions. Always be polite 
and professional.
```

## Step 4: Test Your App

1. Click the **"Test"** button
2. Enter a sample prompt
3. Review the AI's response
4. Adjust settings as needed

## Step 5: Deploy (Pro)

With Nati Pro, deploy your app:

1. Click **"Deploy"**
2. Choose deployment options
3. Get a shareable link
4. Monitor usage and costs

## Example: Simple Chat App

Here's a quick example:

```typescript
// App Configuration
{
  "name": "My Assistant",
  "model": "gpt-4-turbo",
  "systemPrompt": "You are a helpful coding assistant",
  "temperature": 0.7,
  "maxTokens": 2000
}
```

## Common Issues

### API Key Not Working

Make sure:
- Key is valid and active
- Billing is enabled on provider account
- Key has correct permissions

### Model Not Available

Some models require:
- Specific API tier
- Waitlist approval
- Additional credits
