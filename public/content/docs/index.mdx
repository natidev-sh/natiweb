# Welcome to Nati.dev

Nati is a powerful AI app builder that runs locally on your machine. Build AI-powered applications with ease using our desktop app and cloud platform.

## Getting Started

Nati.dev provides two ways to build AI applications:

1. **Desktop App** - Download and run locally on macOS or Windows
2. **Web Platform** - Access your apps and teams from anywhere
